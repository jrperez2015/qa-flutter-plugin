# Repository / Service / DataSource test template

Use when the selected candidate is a class in `lib/src/data/**`, `lib/src/domain/**`, `lib/**/*_repository.dart`, `lib/**/*_service.dart`, `lib/**/*_datasource.dart`, or `lib/**/*_dao.dart`.

## Target path

`{UNIT_TEST_ROOT}/{same_relative_path_as_source}_test.dart`

Example: `lib/src/data/auth_repository.dart` → `test/src/data/auth_repository_test.dart`.

## Analysis before generating

Read the source file and extract:
- `REPO_CLASS` — main class name
- `DEPENDENCIES[]` — constructor params with their types and names
- `METHODS[]` — public methods with signature, return type, `throws` clause

## Template (mocktail)

```dart
// Generated by qa-flutter-unit-generator
import 'package:flutter_test/flutter_test.dart';
import 'package:mocktail/mocktail.dart';
import 'package:{APP_PACKAGE_NAME}/{relative_import}';

{for each dep type:} class Mock{DepType} extends Mock implements {DepType} {}

void main() {
  late {REPO_CLASS} sut;
  {for each dep:} late Mock{DepType} mock{DepName};

  setUp(() {
    {for each dep:} mock{DepName} = Mock{DepType}();
    sut = {REPO_CLASS}({constructor_args_from_mocks});
  });

  {if USES_GETIT:}
  tearDown(() => GetIt.I.reset());

  group('{REPO_CLASS}', () {
    {for each public method M:}
    group('{M.name}', () {
      test('Given valid input, When {M.name} called, Then returns expected', () async {
        // Arrange
        when(() => mock{PrimaryDep}.{method}(any())).thenAnswer((_) async => {expected_fixture});

        // Act
        final result = await sut.{M.name}({fixture_args});

        // Assert
        expect(result, equals({expected_fixture}));
        verify(() => mock{PrimaryDep}.{method}(any())).called(1);
      });

      test('Given dependency throws, When {M.name} called, Then propagates error', () async {
        // Arrange
        when(() => mock{PrimaryDep}.{method}(any())).thenThrow(Exception('boom'));

        // Act & Assert
        expect(() => sut.{M.name}({fixture_args}), throwsA(isA<Exception>()));
      });
    });
  });
}
```

## Template (mockito variant — for `MOCK_LIB == mockito`)

Replace mock class declarations with:

```dart
@GenerateMocks([{DepType1}, {DepType2}, ...])
import '{test_filename}.mocks.dart';
```

Then after writing the file, run:
```bash
dart run build_runner build --delete-conflicting-outputs
```

Mocktail uses `any()` (function call). Mockito uses `any` (no parens). Do not mix.

## Fixtures

Infer fixtures from method return types. For custom domain classes:
- If the class has a simple constructor, emit a best-guess call with default values.
- Otherwise, emit a `// TODO fixture: construct a valid {ClassName}` comment above the `when(...)` line.

## Principles enforced

1. **Mock dependencies, never the SUT.**
2. **Given-When-Then** in every test name.
3. **`GetIt.I.reset()` in tearDown** when the project uses GetIt.
4. **Verify interactions** on the primary dependency (`verify(...).called(N)`).
5. **Always include a failure path** — at least one test where the dependency throws.
